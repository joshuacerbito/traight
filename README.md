# Welcome to Traight (working title)

This is a starter kit for full-stack web application development, and utilizes the following technologies:

1. [Sanity Studio v3](https://www.sanity.io/)
2. [NextJS v13](https://nextjs.org/)
3. [Storybook (w/ CSF 3.0)](https://storybook.js.org/)
4. [TailwindCSS](https://tailwindcss.com/)
5. [HyperUI](https://www.hyperui.dev/)
